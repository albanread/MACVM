MACVM Help & Documentation

This is documentation for the MACVM GUI shell itself — written locally for this project, not part of the original Strongtalk page corpus under reference/pages/. It exists so the "Browse Documentation" link on the Start Page goes somewhere real, and so there is genuine content to click through while testing the shell.
