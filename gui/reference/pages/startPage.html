Welcome to MACVM — inspired by Strongtalk

MACVM builds on the pioneering design of Self and Strongtalk — polymorphic inline caches, type feedback, and deoptimization, and a live hypertext programming environment — reimagined in Rust for Apple Silicon.

This is the MACVM start page. It is designed to be viewed using the MACVM browser, not a regular html browser, because it contains Smalltalk code. Pressing the button on the toolbar (or the toolbar's Home button) will always open this page.

Feel free to modify this page to contain links to your commonly used commands, classes, and applications. This is useful because the MACVM browser, while still primitive in most respects, contains enhancements that let you attach 'do it' code to links (Smalltalk code that is executed when you click on the link). You can also easily imbed Smalltalk applications directly into pages, using the smappl tag. This page contains some examples of these techniques that you can copy and modify. To edit this page, press the button on the toolbar.
